# vue-simple

use vuecli3 to create simple project

## Project structure

``` lua
├── public/                                     # public file
├── src/                                        # source code
│ ├── api/                                      # api file
│ ├── asserts/                                  # 静态资源
│ ├── components/                               # 自定义业务组件
│ ├── config/                                   # 全局配置
│ ├── directive/                                # 自定义组件
│ ├── mock/                                     # 模拟接口数据
│ ├── router/                                   # 路由
│ ├── store/                                    # store
│ ├── view/                                     # 页面
│ ├── lib/                                      # 自定义组件
│ | ├── tools.js                                # 工具方法
│ | ├── util.js                                 # 与业务相关的工具方法
│ ├── App.vue                                   # App 组件
│ ├── main.js                                   # 渲染App 组件
├── .editorconfig                               # 编辑器配置文件 need editorConfig file
├── .eslintrc.js                                # eslint config file
├── .browserslistrc                             # 浏览器兼容配置
├── babel.config.js                             # babel config file
├── vue.config.js                               # vue config file
```

## Project start
```
npm install
```

```
npm run serve
```
